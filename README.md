## ⚙️ Setup Instructions

### Multisim Setup
1.  📥 Download and install [NI Multisim](https://www.ni.com/en-us/support/downloads/software-products/download.multisim.html).
2.  📂 Open `.ms14` files from the `simulations/multisim` directory.

### Octave Setup
1.  📥 Download and install [GNU Octave](https://www.gnu.org/software/octave/download).
2.  ▶️ Run `.m` files from the `simulations/octave` directory.

---

## 🧪 Self-Study Labs

Explore advanced topics with these simulation-based labs using Multisim and Octave.

- **Lab 04: Active Filters** 📉
    - Design and simulate active filters (Low-pass, High-pass, Band-pass) and analyze their frequency response.

- **Lab 05: Instrumentation Amplifiers** 🔬
    - Construct a three-op-amp instrumentation amplifier and investigate its high CMRR for sensor signal amplification.

- **Lab 06: Oscillators and Waveform Generators** 〰️
    - Design and simulate oscillator circuits like Wien bridge (sine) and astable multivibrator (square/triangle).

- **Lab 07: Sensor Signal Conditioning** 🌡️
    - Interface a simulated sensor with a signal conditioning circuit (amplification, filtering, level shifting).

- **Lab 08: Analog-to-Digital Conversion (ADC)** 🔢
    - Simulate a fundamental ADC architecture (e.g., flash or successive-approximation ADC).

- **Lab 09: Digital-to-Analog Conversion (DAC)**  analog
    - Design and simulate an R-2R ladder DAC to convert digital signals back to analog.

- **Lab 10: Basic Data Acquisition (DAQ) System** 📊
    - Integrate sensor, signal conditioning, and ADC components into a simplified DAQ system.
